Metodologia do projeto

Os dados desta plataforma são coletados, várias vezes ao dia, diretamente da API (Interface de Programação de Aplicação) do Twitter. A nossa ferramenta armazena todas as publicações dos perfis da lista "Âncora dos Fatos" na rede social (acesse aqui), que conta com perfis de políticos, candidatos e veículos de imprensa do Paraná, entre outros.

A partir desta base de dados das publicações, identificamos a frequência que alguns termos potencialmente relacionados à desinformação aparecem nos tweets. Os termos utilizados nesta plataforma são: "fake", "corrupto", "roubo" e "fraude".

No entanto, a simples identificação de uma determinada publicação com um dos termos acima citados, não caracteriza, necessariamente, a publicação com teor de desinformação.
